Button Fill Ripple

Spice your buttons up by adding a ripple effect that fills the button!

Below is a standard button. Try hovering over it!

.btn-example, .btn-rpl

Here's some colors you can use:

.btn-rpl, .rpl-blue
.btn-rpl, .rpl-red
.btn-rpl, .rpl-green
.btn-rpl, .rpl-purple

Custom Colors

Want to make your own color? Add the following code to your styles:

.rpl-custom:hover, .rpl-custom:hover:after{     /* Can be anything you want */
    color: #FFF;
    background: #a074d0;                        /* Custom color here */
    border-color: #a074d0;                      /* Make it the same as the above color */
}

Then add the following classes to your button:

.rpl-btn, .rpl-custom
